“Serenity AI Chat: Aplicación Web Inteligente para el Análisis de Sentimientos y Lenguaje Tóxico”

El proyecto "Serenity AI Chat" es una aplicación web que combina tecnologías modernas y herramientas de inteligencia artificial para ofrecer un análisis automático del sentimiento y la toxicidad de textos ingresados por los usuarios. Desarrollado en el marco del curso Desarrollo de Aplicaciones Web, este sistema demuestra la integración de un backend en Python (Flask) y un frontend en React.js, conectados a una base de datos MySQL y potenciados por bibliotecas de NLP como TextBlob y la Perspective API de Google.El objetivo principal fue brindar una herramienta funcional, intuitiva y segura que permita a los usuarios reflexionar sobre su comunicación escrita, clasificando sus mensajes como positivos, negativos, neutros o potencialmente ofensivos.

Arquitectura Cliente-Servidor

  • • Separación lógica del frontend (React) y backend (Flask).
  • • Comunicación asincrónica vía API RESTful.

Inteligencia Artificial

  • • TextBlob para análisis de sentimiento (polaridad emocional).
  • • Perspective API para detección de toxicidad, insultos y amenazas en el texto.

Autenticación y Seguridad

  • • Uso de JSON Web Tokens (JWT) para proteger rutas sensibles y mantener sesiones seguras.
  • • Cifrado de contraseñas y validación de datos para prevenir ataques.

Frontend Moderno

  • • Desarrollo con React.js y diseño responsive con HTML5, CSS3 y JavaScript.
  • • Interfaz amigable, accesible y estéticamente coherente con la temática emocional del proyecto.

Base de Datos y ORM

  • • Gestión de usuarios con MySQL.
  • • Uso de SQLAlchemy como ORM para simplificar las operaciones de base de datos.

Experiencia de Usuario (UX/UI)

  • • Paleta de colores pastel, tipografías legibles y componentes interactivos.
  • • Retroalimentación visual inmediata y mensajes interpretativos basados en el análisis IA.

Pruebas del Sistema

  • • Pruebas unitarias, funcionales y de integración aseguraron estabilidad, velocidad y precisión.

Reflexión

¿Qué Aprendí?

Aprendí a desarrollar una aplicación web completa integrando múltiples tecnologías modernas como Flask, React.js, MySQL y bibliotecas de inteligencia artificial como TextBlob y Perspective API. Comprendí cómo funciona la autenticación segura mediante JWT, cómo estructurar una API RESTful, y cómo aplicar principios de diseño UX/UI centrados en el usuario. También aprendí a utilizar IA para el análisis de sentimientos y lenguaje tóxico, reconociendo su potencial ético y social en la mejora de la comunicación digital.
