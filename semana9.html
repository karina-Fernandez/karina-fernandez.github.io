Desarrollo Backend Java Spring

El desarrollo backend es una parte fundamental en la creación de aplicaciones web, ya que se encarga de la lógica del servidor, la gestión de datos y la comunicación entre el cliente y el servidor. Para ello, se utilizan diferentes arquitecturas, servidores, lenguajes y herramientas que permiten construir sistemas eficientes, seguros y escalables.

Arquitecturas de Aplicaciones Web

  • • Cliente/Servidor Clásica (MPA): Cada vista es una página distinta (HTML, CSS, JS).
  • • Multicapa (MPA con MVC): Divide el sistema en capas con responsabilidades específicas.
  • • SPA (Single Page Application): Una sola página HTML; usa JavaScript para simular navegación, solicitando solo datos al servidor.
  • • Híbridas SPA/MPA: Mezcla funcionalidades de ambos enfoques. Ej. Next.js.
  • • Arquitectura Hexagonal: Usa puertos y adaptadores para desacoplar componentes y facilitar pruebas.

Servidores Web

  • • Servidor Web:Software que atiende solicitudes HTTP y entrega archivos (HTML, CSS, JS).
  • • Ejemplos de software: Apache, Tomcat, NGINX, Node.js.
  • • Tipos de Hosting:Compartido, VPS Dedicado Administrado, Cloud (en la nube)

    • Desarrollo Backend

  • • Lenguajes comunes: Java, PHP, Node.js, Python.
  • • Gestores de base de datos: MySQL, PostgreSQL, SQL Server.
  • • Herramientas
  • - Maven: Automatiza compilación, pruebas, empaquetado y despliegue. Usa el archivo pom.xml
  • - Gradle: Similar a Maven, usa scripts en Groovy.

    • Autenticación con Tokens

  • • Token: Dato cifrado que valida identidad sin reenviar credenciales.
  • • Tipos: JWT (Json Wer Token), Oauth, Saml

    • Entorno de Desarrollo

  • • Instalación requerida:
  • - JDK (Java Development Kit)
  • - IntelliJ IDEA (IDE de desarrollo)
  • - Apache Tomcat (servidor web para Java)

    Reflexión

    ¿Qué Aprendí?

    Aprendí los fundamentos del desarrollo backend, incluyendo las diferentes arquitecturas web como MPA, SPA e híbridas, y su utilidad en la creación de aplicaciones modernas. Entendí la función de los servidores web, los tipos de hosting y el rol del DNS en la publicación de sitios. También aprendí a configurar servidores como Apache y Tomcat, y a utilizar herramientas como Maven y Gradle para gestionar proyectos y sus dependencias.
