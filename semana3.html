HTML Básico y Avanzado, CSS Básico y Avanzado

Fundamentos y técnicas avanzadas de HTML para estructura semántica y CSS para diseño, layout y presentación visual en el desarrollo web moderno.

Temas Aprendidos

HTML Básico - Estructura y Semántica

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Aprendí la estructura básica con elementos como <html>, <head>, <body>, etiquetas semánticas como <header>, <nav>, <main>, <section>, <article>, <footer>, y elementos de contenido como párrafos, listas, enlaces, imágenes y formularios. La semántica correcta mejora la accesibilidad y SEO.

HTML Avanzado - Formularios y Multimedia

Exploré elementos avanzados como formularios complejos con validación HTML5, tipos de input especializados (email, date, range, color), elementos multimedia (<video>, <audio>, <canvas>), APIs modernas como Web Storage, Geolocation, y técnicas de optimización como lazy loading de imágenes, meta tags para SEO, y structured data para rich snippets.

CSS Básico - Estilos y Selectores

CSS (Cascading Style Sheets) controla la presentación visual de HTML. Dominé selectores básicos (elemento, clase, ID), propiedades fundamentales (color, font, margin, padding, border), el modelo de caja (box model), unidades de medida (px, em, rem, %, vh, vw), y conceptos de cascada, especificidad y herencia que determinan cómo se aplican los estilos.

CSS Avanzado - Layout y Animaciones

Implementé técnicas avanzadas como Flexbox para layouts unidimensionales, CSS Grid para layouts bidimensionales complejos, posicionamiento avanzado (relative, absolute, fixed, sticky), transformaciones 2D y 3D, animaciones con @keyframes, transiciones suaves, variables CSS (custom properties), y técnicas de responsive design con media queries.

Reflexión

¿Qué aprendí?

  • • A crear la estructura básica de una página web usando HTML.
  • • A usar etiquetas importantes como header, nav, section y footer.
  • • A escribir formularios simples con campos como texto, correo y botón.
  • • Animaciones y transiciones CSS para mejorar la interactividad
  • • A aplicar estilos con CSS para cambiar colores, tamaños y márgenes.
  • • A hacer que mi página se vea bien en diferentes tamaños de pantalla.
  • • Optimización de rendimiento y mejores prácticas de desarrollo web

¿Cómo aprendí?

  • • Construcción progresiva de proyectos desde básico hasta avanzado
  • • Experimentación práctica con diferentes propiedades y valores CSS
  • • Análisis de sitios web existentes para entender técnicas implementadas
  • • Viendo videos que explican cómo usar HTML y CSS desde cero.
  • • Observando cómo están hechas otras páginas web.
  • • Usando el navegador para ver los cambios y ajustar estilos.
