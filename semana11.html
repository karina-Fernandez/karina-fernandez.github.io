Desarrollo Práctico de Aplicaciones Web en Java con JSP y Spring Boot

La parte práctica del desarrollo backend con Java permite aplicar directamente los conceptos aprendidos mediante la creación de aplicaciones web, APIs RESTful y la integración de herramientas como Tomcat, Maven, IntelliJ IDEA y Spring Boot. Esta práctica proporciona la experiencia necesaria para entender cómo funciona una aplicación Java en un entorno real.

Parte Práctica con Conceptos

1. Instalación del Entorno de Desarrollo

  • • 1.Instalación del Entorno de Desarrollo
  • - JDK OpenJDK: Instalación del kit de desarrollo Java para compilar y ejecutar aplicaciones.
  • - Tomcat: Servidor para desplegar aplicaciones web Java.
  • • IntelliJ IDEA: IDE utilizado para programar en Java y gestionar proyectos con Maven.

2. Creación de Aplicaciones JSP

  • • Aplicación básica que muestra la fecha y hora actual.
  • • Aplicación con formulario que calcula el factorial de un número utilizando el método GET.
  • • Uso de objetos implícitos (request, session) para gestionar datos entre usuario y servidor.

    • 3. Desarrollo de una API REST con Spring Boot

    • • Generación del proyecto desde start.spring.io .
  • • Configuración del archivo application.properties para conectar con MySQL.
  • • Creación de paquetes:
  • - models: contiene las entidades (por ejemplo, Docentes.java).
  • - repositories: interfaces JPA para acceder a la BD.
  • - controllers: manejo de las solicitudes HTTP.
  • • Implementación de operaciones CRUD completas:
  • - Crear, leer, actualizar y eliminar registros.
  • - Consultas por ciudad, años de experiencia y edad promedio.
  • - Validaciones con reglas específicas (correo válido, fecha anterior, etc.).
  • - Documentación con Swagger.
  • - Paginación con parámetros page y size

    • 4. Pruebas con Postman

  • • Se utilizaron endpoints REST para probar las operaciones CRUD.
  • - Se enviaron peticiones POST, GET, PUT y DELETE para validar el correcto funcionamiento del backend.
  • - Se verificó la respuesta en formato JSON y el estado HTTP.

    Reflexión

    ¿Qué Aprendí?

    Aprendí a desarrollar aplicaciones web dinámicas utilizando tecnologías Java como JSP, Jakarta EE y Spring Framework. Comprendí el uso de servidores como Apach e Tomcat, el patrón MVC, la configuración de proyectos con Maven, y la implementación de APIs RESTful con operaciones avanzadas como validaciones, filtros y paginación.

    ¿Cómo Aprendí?

    Lo aprendí a través de una combinación de teoría clara y ejercicios prácticos. Instalé herramientas como JDK, IntelliJ IDEA y Tomcat, configuré proyectos con Spring Boot, y probé las funcionalidades de la API usando Postman. Esta metodología me permitió asimilar los conceptos de manera progresiva, con una experiencia de aprendizaje práctica y aplicada.
