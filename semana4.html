CSS con Bootstrap, CSS con Tailwind y JavaScript Básico

Frameworks CSS modernos para desarrollo rápido y eficiente, junto con fundamentos de JavaScript para agregar interactividad y dinamismo a las aplicaciones web.

Temas Aprendidos

CSS con Bootstrap - Framework Component-Based

Bootstrap es un framework CSS que proporciona componentes prediseñados y un sistema de grid responsivo. Aprendí a utilizar clases utilitarias, componentes como navbar, cards, modals, carousel, y el sistema de grid de 12 columnas. Bootstrap acelera el desarrollo proporcionando estilos consistentes y componentes JavaScript integrados, ideal para prototipos rápidos y proyectos que requieren un diseño profesional sin personalización extensiva.

CSS con Tailwind - Utility-First Approach

Tailwind CSS es un framework utility-first que proporciona clases de bajo nivel para construir diseños personalizados. Dominé conceptos como spacing (m-, p-), sizing (w-, h-), colors, typography, flexbox utilities, y responsive design con prefijos (sm:, md:, lg:). Tailwind permite mayor control y personalización, resultando en diseños únicos y CSS optimizado mediante purging de clases no utilizadas.

JavaScript Básico - Fundamentos del Lenguaje

JavaScript es el lenguaje de programación que añade interactividad a las páginas web. Aprendí sintaxis básica, variables (var, let, const), tipos de datos (string, number, boolean, object, array), operadores, estructuras de control (if/else, loops), funciones, y scope. También exploré conceptos fundamentales como hoisting, closures, y la diferencia entre declaración de funciones y expresiones de función.

DOM Manipulation y Event Handling

El DOM (Document Object Model) permite a JavaScript interactuar con elementos HTML. Aprendí métodos de selección (getElementById, querySelector), manipulación de contenido (innerHTML, textContent), modificación de atributos y estilos, creación y eliminación de elementos. Implementé event listeners para manejar interacciones del usuario como clicks, form submissions, y keyboard events, creando experiencias web dinámicas.

Reflexión

¿Qué aprendí?

  • • Diferencias entre frameworks CSS: Bootstrap (component-based) vs Tailwind (utility-first)
  • • Ventajas y desventajas de cada enfoque según el tipo de proyecto
  • • Fundamentos sólidos de JavaScript: sintaxis, tipos de datos, y control de flujo
  • • Manipulación del DOM para crear interfaces dinámicas e interactivas
  • • Event handling y mejores prácticas para experiencia de usuario
  • • Persistencia de datos con localStorage y APIs del navegador

¿Cómo aprendí?

  • • Comparación práctica entre Bootstrap y Tailwind en proyectos similares
  • • Construcción incremental de aplicaciones JavaScript desde cero
  • • Debugging sistemático usando herramientas de desarrollo del navegador
  • • Experimentación con diferentes patrones de código y arquitecturas
  • • Análisis de código fuente de librerías para entender implementaciones
  • • Iteración continua basada en feedback de usabilidad y rendimiento
